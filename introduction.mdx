---

FitWave is a web application built to eliminate the friction in gym operations. Members can self-serve — registering an account, browsing available classes, and booking sessions without staff involvement. Administrators get a dedicated dashboard to manage every aspect of the gym: trainers, class schedules, bookings, and user accounts.

The frontend is an Angular 21 single-page application styled with TailwindCSS v4 and DaisyUI v5. It communicates with a REST API backend running at `http://localhost:8080`.

## User roles

FitWave has two roles that determine which routes and features are accessible after login.

| Role | Default route | Access |
|---|---|---|
| Member | `/user-dashboard` | Browse classes, book and cancel sessions, view personal bookings |
| Administrator | `/admin-dashboard` | All member access plus trainer, schedule, class, and user management |

<Note>
  Role assignment is handled server-side. The `roles` field on the authenticated user object determines which dashboard the Angular router redirects to after login.
</Note>
