Trainers are the professionals who lead gym classes at FitWave. Before you can create a class, you must have at least one trainer registered in the system.

Trainer management lives inside the collapsible **GestiÃ³n de Entrenadores** section on the [Admin Dashboard](/admin-guide/admin-dashboard).

## The Trainer interface

```typescript
export interface Trainer {
  id?: number;      // Assigned by the backend; omit when creating
  name: string;     // Full name of the trainer
  specialty: string; // One of the accepted specialty values
  bio: string;      // Biographical description
}
```

### List all trainers

```bash
GET /api/trainers
```

Returns an array of `Trainer` objects.

### Delete a trainer

```bash
DELETE /api/trainers/:id
```

Permanently removes the trainer with the given `id`. Returns `204 No Content` on success.

## Registered Staff table

The table beneath the form lists all trainers currently in the system with the following columns:

| Column | Description |
|--------|-------------|
| ID | Auto-assigned numeric identifier (monospace format) |
| Entrenador | Full name of the trainer |
| Especialidad | Specialty badge |
| Acciones | View details and delete buttons |

### Viewing trainer details

Click the eye icon in the **Acciones** column to open a modal showing the trainer's full name, specialty badge, and complete bio.

### Deleting a trainer

<Warning>
  Deleting a trainer is permanent and cannot be undone. Any classes currently assigned to this trainer will retain the trainer reference in the database, but the trainer will no longer appear in the system.
</Warning>

Click the trash icon in the **Acciones** column. A confirmation modal will appear showing the trainer's name and warning that the action cannot be undone. Click **SÃ­, eliminar** to proceed or **No, cancelar** to dismiss.
